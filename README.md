
### Project Guide

- setup env
```bash
    cp .env.example .env
```
- Install dependencies

```bash
  npm install
```

- start project 
```bash
  npm run start
```
- start test 
```bash
  npm run test
```
- vercel setup
```
  `npm run vercel-build` or `npm run build` or `npm run start:dev`
```
